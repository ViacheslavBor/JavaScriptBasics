JavaScript Basics

JavaScripl is a cross-platform, object-oriented script language. JavaScript is extremely popular for a variety of reasons.

  • Variable Declaration JavaScript variables are containers for storing data values - imagine a cup you fill with cofee, the cup holds the coffee, a variable holds a value. All JavaScript variables must be identified with unique names. These unique names are called identifiers. var x;
  • Variable Assignment Assignment operators assign values to JavaScript variables - our cup can now have coffee poured in it, giving our variable a value to hold. The = assignment operator assigns a value to a variable. var x = 10;
